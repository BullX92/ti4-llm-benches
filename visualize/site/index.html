Workflow

  1. Add or refresh an Ollama-compatible provider config.
  2. Load models from each configured provider; the pipeline runs missing or invalid testcase/model combinations automatically.
  3. Define testcases with prompt + expected key/value results.
  4. Run the matrix; unchanged pairs are served from SQLite cache.
